## рҹ”‘ **5 CAMBIOS ARQUITECTГ“NICOS CLAVE**

### **1пёҸвғЈ STOCK MГҚNIMO (CRГҚTICO PARA COMPRAS)**

**Problema actual**: No existe. Compramos sin saber cuГЎndo.

**SoluciГіn**:
```typescript
// Tabla recursos - agregar:
interface Recurso {
  id: number;
  codigo: string;
  nombre: string;
  categoria_id: number;
  unidad_medida_id: number;
  
  // NUEVO: Control de inventario
  stock_minimo: number;  // ВҝCuГЎndo comprar?
  stock_maximo?: number; // ВҝCuГЎndo dejar de comprar?
  
  // Precios
  precio_unitario: number;
  proveedor_id?: number;
  
  // Estado
  activo: boolean;
}

// Ejemplo:
{
  nombre: "Cemento Portland 50kg",
  stock_minimo: 100,  // Si llega a 100, ВЎCOMPRA!
  stock_maximo: 300,  // No compres si tienes 300+
  precio_unitario: 45.50,
  proveedor_id: 5
}
```

**Impacto**: 
- вң… Nunca mГЎs "se acabГі el cemento"
- вң… No compras de mГЎs (evita overstock)
- вң… Sistema alerta automГЎticamente

### **2пёҸвғЈ COSTO TOTAL POR RECURSO (VITAL PARA PRESUPUESTOS)**

**Problema actual**: No existe. No saben cuГЎnto gastaron en materiales.

**SoluciГіn**:
```typescript
// Vista materializada: costo_actual
interface RecursoConCosto {
  id: number;
  codigo: string;
  nombre: string;
  stock_actual: number;
  precio_unitario: number;
  costo_total: number;  // stock_actual * precio_unitario
  
  // Por frente:
  costo_en_frentes: {
    frente_id: number;
    frente_nombre: string;
    stock_en_frente: number;
    costo_en_frente: number;  // stock * precio
  }[]
}

// Ejemplo:
{
  nombre: "Hierro Гҳ 8mm",
  stock_actual: 120,
  precio_unitario: 32,
  costo_total: 3840,  // 120 * 32
  
  costo_en_frentes: [
    { frente: "Casa PeГұa", stock: 50, costo: 1600 },
    { frente: "Playa Resort", stock: 70, costo: 2240 }
  ]
}
```

**Impacto**:
- вң… Gerente sabe: "InvertГӯ $12,400 en materiales esta semana"
- вң… Cliente pregunta: "ВҝCuГЎnto gastaste en Frente A?" вҶ’ Respuesta inmediata
- вң… Presupuesto actualizado en tiempo real
